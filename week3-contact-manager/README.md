# Contact Management System

## Project Description
This is a simple contact management system built using Python.
The project uses functions and dictionaries to store and manage contact details.

## Features
- Add new contacts
- Search contacts by name
- Update contact information
- Delete contacts
- View all saved contacts
- Save data using JSON file

## Concepts Used
- Functions
- Dictionaries
- File handling
- String methods
- Input validation

## How to Run
```bash
python contacts_manager.py
