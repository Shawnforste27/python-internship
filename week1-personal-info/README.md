
# Personal Information Manager

## Project Description
This is my first Python project for Week 1 of the Python Internship.
The program stores personal information, takes input from the user,
and displays the information in a formatted way.

## What This Program Does
- Stores personal details like name, age, city, and hobby
- Takes user input for favorite food and favorite color
- Checks if user input is empty
- Calculates age in months
- Displays all information clearly
