# Personal Information Manager

## Project Description
This is my first Python project. It stores personal information and takes input from the user.

## What I Learned
- How to use variables
- How to take input from user
- How to print output
- Basic input validation
- Simple calculations

## How to Run the Program
1. Install Python
2. Open terminal
3. Go to project folder
4. Run: python personal_info.py

## Features
- Stores name, age, city and hobby
- Takes favorite food and color from user
- Checks empty input
- Calculates age in months
